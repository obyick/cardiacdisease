Análise de Pacientes com Risco Cardíaco

Descoberta de padrões através de Machine Learning não supervisionado.

Análise de Cluster Uso de K-Means para segmentar perfis de pacientes.
Visualização 2D Redução de dimensionalidade com PCA para visualização.
Stack Moderno Construído com Angular, TypeScript e Chart.js.

Visão Geral e Objetivos

Este projeto consiste em um dashboard interativo, desenvolvido para explorar um conjunto de dados clínicos sobre doenças cardíacas. O objetivo central não é prever um diagnóstico, mas sim aplicar técnicas de aprendizado de máquina não supervisionado para identificar e caracterizar grupos distintos de pacientes (clusters) com perfis semelhantes.

A aplicação busca responder a perguntas estratégicas, como:

  • Existem agrupamentos naturais de pacientes com base em suas características clínicas?
  • Quais são os perfis que definem cada um desses grupos?
  • Como esses perfis se distribuem em relação a fatores demográficos e sintomáticos?

Metodologia e Pipeline de Dados

O núcleo do projeto é um serviço de análise que executa um pipeline de Machine Learning, implementado em TypeScript:

  • Pré-processamento: Os dados são normalizados através da padronização (Z-score). Este passo é fundamental para que o algoritmo K-Means trate todas as características com igual importância, independentemente de sua escala original.
  • Clusterização (K-Means): O algoritmo K-Means foi implementado para segmentar os pacientes em 3 clusters distintos, agrupando-os com base em 10 características clínicas, como idade, pressão arterial, colesterol e frequência cardíaca.
  • Redução de Dimensionalidade (PCA): Para visualizar os clusters, a técnica de Análise de Componentes Principais (PCA) foi utilizada para reduzir as 10 dimensões dos dados a apenas duas, permitindo a plotagem em um gráfico de dispersão 2D sem perda significativa de informação sobre a variância.

Arquitetura e Tecnologias

A aplicação foi desenvolvida com tecnologias modernas, visando performance, manutenibilidade e uma rica experiência de usuário:

  • Framework Frontend: Angular, com a lógica dos componentes escrita em TypeScript.
  • Visualização de Dados: Gráficos dinâmicos e interativos renderizados com a biblioteca Chart.js (via ng2-charts).
  • Computação Matemática: A biblioteca Math.js foi utilizada para as operações de álgebra linear necessárias nas implementações customizadas de K-Means e PCA.
  • Estilização: A interface foi construída com SCSS, utilizando variáveis, Flexbox e Grid Layout para criar um design responsivo, temático (dark mode) e com elementos em "glassmorphism".

Conclusão

Este projeto demonstra com sucesso um pipeline completo de ciência de dados, transformando dados brutos em insights visuais e acionáveis. Ao identificar arquétipos de pacientes, a ferramenta serve como um poderoso exemplo de como o aprendizado não supervisionado pode ser usado para análise exploratória e descoberta de conhecimento em datasets complexos no campo da saúde.
